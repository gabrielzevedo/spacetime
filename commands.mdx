# Commands used to create this project

## Setup
```bash
npm init -y
```

## Typescript
```bash
npm i typescript -D
npm install @types/node -D
npx tsc --init
npm install tsx -D
```

## Fastify
```bash
npm i fastify
```

## ESLint
```bash
npm install eslint -D
npm i @rocketseat/eslint-config -D
```

## Prisma
```bash
npm i prisma -D
npx prisma init --datasource-provider SQLite
npx prisma studio
npm i @prisma/client
```

After create user table
```bash
npx prisma migrate dev
```

## Next.js
```bash
npx create-next-app@latest web --use-npm
```

## Icons
```bash
npm i lucide-react
```

## TailwindCSS + Prettier
Install this in web and mobile folders.

```bash
npm i @rocketseat/eslint-config -D
npm i prettier-plugin-tailwindcss -D
```

## Mobile
```bash
npx create-expo-app mobile
npm i nativewind
npm i tailwindcss -D
npx tailwindcss init
```

Follow instructions from [NativeWind](https://www.nativewind.dev/quick-starts/expo)

## Mobile fonts
```bash
npx expo install @expo-google-fonts/roboto @expo-google-fonts/bai-jamjuree expo-font

## Mobile SVG
Follow instructions from [react-native-svg-transformer](https://github.com/kristerkari/react-native-svg-transformer)

```bash
npx expo install react-native-svg
npm i -D react-native-svg-transformer
```

## Prisma configs
```bash
npx prisma migrate reset
npx prisma migrate dev
```

## Data validation
```bash
npm i zod
```

## Cors
```bash
npm i @fastify/cors
```

## Dotenv
```bash
npm i dotenv -D
```

## Axios
```bash
npm i axios
```

## JWT
```bash
npm i @fastify/jwt
```

In web:
```bash
npm i jwt-decode
```

## Mobile session
Follow instructions from [Expo AuthSession](https://docs.expo.dev/versions/latest/sdk/auth-session/)

```bash
npx expo install expo-auth-session expo-crypto
npx expo install expo-secure-store
```
